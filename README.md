# autobitbucket

Script para configurar auto-deploy com Serverpilot e Bitbucket.

  - Serve para qualquer servidor
  - Pode ser modificado para outros fins
  - Codigo Aberto



### Utilização

```sh
$ git clone https://github.com/shirleyson/autobitbucket.git kaiser
$ cd kaiser
$ chmod 775 kaiser.sh
$ ./kaiser.sh
```
